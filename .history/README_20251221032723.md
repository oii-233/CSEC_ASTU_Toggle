# Interactive Lamp Toggle 💡

A simple **lamp toggle** project built with **HTML, CSS, and JavaScript**.

Click the button to switch the lamp **ON/OFF**, changing the background and lamp glow.

---

## Features

- Lamp ON/OFF toggle
- Background changes: dark → light
- Lamp icon glows when ON
- Responsive design
- Smooth transitions

---

## Technologies

- HTML5
- CSS3 (Flexbox, Gradients, Transitions)
- JavaScript (DOM, Event Listeners)
- [Lucide Icons](https://lucide.dev)

## How to Run

1. Open `index.html` in a browser.
2. Click **Switch On** to light the lamp.
3. Click **Switch Off** to turn it off.
